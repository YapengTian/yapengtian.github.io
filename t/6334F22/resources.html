Project

Students are expected to work in a team (up to 4 members) on projects related to VR and present their work at the end of the semester.

Project Proposal Instructions:

The goal of this course project is to build a prototype system for virtual reality or augmented reality. Please read the Project Description and follow the Proposal instruction to prepare your project proposal.

(Optional) Besides designing VR systems, another option is to do an exploratory research project. The goal of this research-oriented course project is to work on one research topic related to virtual reality or augmented reality. Please read the Project Description and follow the Proposal instruction to prepare your project proposal.
